📌 Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on the Student Performance dataset to
understand the key factors influencing academic outcomes. By analyzing variables such as study time, 
parental education, and socio-economic background, the project aims to uncover meaningful insights and 
visualize patterns that can help in predicting and improving student success.

Source:- student-por.csv
